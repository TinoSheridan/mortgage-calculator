OBMMI Ticker Tape Integration

About OBMMI (Optimal Blue Mortgage Market Indices)

The ticker tape banner above displays real-time mortgage market data from the Optimal Blue Mortgage Market Indices (OBMMI). This data represents approximately 35% of the U.S. mortgage market and provides more accurate rate information than traditional advertised rates because it's based on actual loan locks.

  • Real-time data from actual loan locks (not advertised rates)
  • Covers ~35% of the U.S. mortgage market
  • Updates multiple times daily
  • Includes major loan types: 30Y Fixed, 15Y Fixed, FHA, VA, Jumbo
  • Mobile-responsive ticker tape design
  • Minimal screen real estate usage

Implementation Features

  • Scrolling ticker tape animation
  • Hover to pause functionality
  • Mobile-optimized responsive design
  • Accessibility features (reduced motion support)
  • Automatic refresh every 5 minutes
  • Fallback data when widget is unavailable
  • Proper Optimal Blue attribution

Integration Instructions

To integrate the OBMMI widget on your website:

  1. Include the CSS and JavaScript files in your HTML
  2. Obtain the official OBMMI widget script from Optimal Blue
  3. Update the widget script URL in the JavaScript file
  4. Customize the styling to match your brand

Required Files:

<link rel="stylesheet" href="static/css/obmmi-ticker.css"> <script src="static/js/obmmi-ticker.js"></script>

Mobile Optimization

The ticker tape is specifically designed to minimize screen real estate usage on mobile devices:

  • Compact height: 30-40px depending on screen size
  • Faster scrolling speed on smaller screens
  • Optimized font sizes for readability
  • Touch-friendly hover states
  • Reduced motion support for accessibility

Customization Options

The ticker can be customized for your branding:

  • Color scheme and gradients
  • Animation speed and direction
  • Font family and sizes
  • Loan types to display
  • Positioning (top or bottom of page)
  • Refresh intervals
